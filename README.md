# Real-Id-for-Skyrim-Wiki
Replaces partial Id, at Skyrim Wikis, by their real Dawnguard/Hearthfire/Dragonborn Id.
### Works for https://en.uesp.net/wiki and https://elderscrolls.fandom.com/wiki.
